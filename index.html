Installing Python

To install python go to https://www.python.org/ and download the latest version for your os. After the download is complected install the package. If on windows make sure to select add to path while installing

on ubuntu linux you could just apt-get install python

you would also need a text editor and terminal or powershell to write and run your python programs, alternatively you could use repel.it to skip the installation and learn python on your web browser

Hello world

Now that you have a text editor and python ready, it's time to write your first program - the Famous hello world program which is what all programers write first - it's kinda a tradition

open your text editor and make a new file, type print("hello world") and save the file with a .py file extension

open a terminal window in the same folder and type python {filename} change filename to the name you save the file with

you would see the words hello world print out, congratulations you just wrote your first python program
